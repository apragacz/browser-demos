SVG Text Gradient examples

Here are some examples of using SVG linear gradient over SVG tspan (or text - in the last example). We are using gradientUnits="userSpaceOnUse" because setting to "objectBoundingBox" gives more chaotic results (the "object" bounding box sometimes seems to be the text bbox, not tspan bbox). We assume that the example works (does not fail) iff the sharp gradient transitions are aligned with the dark gray horizontal paths under the text.
