SVG Arabic Text examples

Here are some examples of using arabic with SVG text (textpath) elements.

Working examples

These should work on Chrome & Firefox browser.

  1. Text, the characters should display as 'connected'.
  2. Text, tspanned. The characters should display as 'separated'.
  3. Text, tspanned with zero-width joiner. The characters should display as 'connected'

Failing examples

These examples will fail on webkit based browsers (like chromium 35.0, chrome and safari) - the arabic characters will be rendered as 'separated' instead of 'connected'. They will not fail on Firefox 25.0 (in version 29 text is not bent, but that is already reported bug).

  1. Text on horizontal line path
  2. Text on nonlinear bezier path
  3. Text on horizontal line path, tspanned with zero-width joiner
  4. Text on nonlinear bezier path, tspanned with zero-width joiner
